# House Price Prediction (Boston Housing Dataset)

A beginner-friendly machine learning project that predicts house prices using the Boston Housing dataset.  
This project demonstrates data cleaning, exploratory data analysis (EDA), model training, evaluation, and visualization.

## Features
- Load and explore the Boston Housing dataset
- Perform EDA and correlation analysis
- Train a Linear Regression model
- Evaluate performance using MAE, MSE, and RÂ²
- Visualize Actual vs Predicted prices

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
